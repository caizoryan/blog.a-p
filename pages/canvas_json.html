In the implementation, we're going to use this spec . What is this you ask? It is a spec for a file format for infinite canvas apps. The benefit of using this is that if we make our app import and export this file format; then the file can be used in other apps such as Obsidian or Kinopio. Open file formats facilitate interoperability. You can read a little bit about it here for more details.
